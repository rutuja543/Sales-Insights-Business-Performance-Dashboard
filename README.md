📊 Sales Insights & Business Performance Dashboard

An interactive Power BI dashboard designed to analyze sales performance, identify trends, and provide actionable business insights.

🚀 Project Overview

This project focuses on:

Understanding overall sales performance
Analyzing category-wise contribution
Comparing regional sales
Tracking monthly sales trends

It helps businesses make data-driven decisions to improve revenue and performance.

📌 Key Features
📊 KPI Cards (Total Sales, Profit, Orders, Quantity)
📈 Monthly Sales Trend Analysis
🛍 Category-wise Sales Distribution
🌍 Regional Sales Comparison
📉 Category Trends Over Time
🎯 Interactive Filters for better exploration
🛠 Tools & Technologies
Power BI (Data Visualization)
DAX (Basic calculations)
Excel / CSV Dataset

📊 Key Insights
Certain categories contribute more to overall revenue
Sales trends fluctuate across months
Regional performance varies significantly
Identifies high-performing and low-performing segments
📁 Dataset
Superstore Sales Dataset
Fields used:
Sales
Profit
Quantity
Category
Region
Order Date
🎯 Business Use Case

This dashboard can be used by:

Sales Teams → Track performance
Managers → Identify growth opportunities
Executives → Make strategic decisions
